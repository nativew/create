# Create 🎨
Command-line tool for [Native Web](https://github.com/nativew/nativeweb).

## ✨ Features
- **Init**. Start a project with [Start 🧨](https://github.com/nativew/start).

## 🚀 Start
```zsh
# Create a new project
npm init nativeweb

# And run it 👟
npm start
```
